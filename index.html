**To connect a Repository to the local file for the first time**:
⇒ First of all, make sure that CMD is in the right directory
⇒ It is advised to create the Directory in C Drive
⇒ To check go to the directory using “cd \“ command
⇒ Then follow the commands above the connect a repository

**To push newly added files**:
⇒ Only three steps need to be done:
1. git add .
2. git commit -m”add a comment”
3. git push
